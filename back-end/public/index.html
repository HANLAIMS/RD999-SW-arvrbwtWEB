Get Ship list get all
curl localhost:3000/pg-shiplist -H 'Authorization: Bearer token'
Get Ship list get one
curl localhost:3000/pg-shiplist/:Ship-ID -H 'Authorization: Bearer token'
Get Ship data on total of oprations
curl -X POST localhost:3000/pg-shipdata/:Ship-ID -H 'Authorization: Bearer token'
Get Tag name according to operation mode
curl -X POST localhost:3000/pg-shipdata-getcol/:Ship-ID -H 'Authorization: Bearer token' -d "opmode=string"
Get Data based on tag name and uptime
curl -X POST localhost:3000/pg-shipdata-opdata/:Ship-ID -H 'Authorization: Bearer token' -d "opmode=string,columnlist=string,startdate=string,enddate=string"
Get Average data based on tag name and uptime
curl -X POST localhost:3000/pg-shipdata-avgopdata/:Ship-ID -H 'Authorization: Bearer token' -d "opmode=string,columnlist=string,startdate=string,enddate=string"
